# Quadratic Equation Solver and Parabola Plotter

This app, written in Kotlin, calculates solutions to quadratic equations and plots the corresponding parabola. 

## Features

- **Solve Quadratic Equations**: It computes the roots of quadratic equations in the form `ax^2 + bx + c = 0`.
- **Plot Parabola**: The app visualizes the graph of the quadratic equation, showing the parabola on a coordinate plane.

## Usage

1. **Input**: The app takes the coefficients `a`, `b`, and `c` as input.
2. **Calculation**: It calculates the discriminant (Δ) and then computes the real or complex roots.
3. **Plotting**: The app generates a visual plot of the parabola, showing the vertex and the axis of symmetry.

## How to Run

1. Clone the repository to your local machine.
2. Open the project in Android Studio.
3. Build and run the app on your device or emulator.

## Example

For the quadratic equation `2x^2 + 3x - 2 = 0`:

- **Roots**: The app will compute and display the real roots.
- **Parabola**: The app will plot the parabola for this equation.
